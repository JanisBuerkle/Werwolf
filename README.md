# 🐺 Werwolf WPF App 🎮
Welcome to the **Werwolf WPF App**! This is a desktop application built using WPF (Windows Presentation Foundation) to bring the thrilling game of Werwolf to your PC. Enjoy an immersive experience with friends as you play a digital version of this classic party game.

## 🌟 Features
- 🐺 **Role Variety**: Play with a range of classic roles, including:
  - **Villagers**: 👨‍🌾 Work together to uncover the werewolves.
  - **Werewolves**: 🐺 Secretly eliminate villagers without being discovered.
  - **Special Roles**: 🔮 Unique abilities to add depth and strategy to the game.

- 📊 **Game Management**: Automatic role assignment and game progression, making it easy to focus on strategy and fun.

- 🎨 **User-Friendly Interface**: Built with WPF for a smooth and responsive user experience on Windows devices.

## 📚 Technologies Used

- **WPF (Windows Presentation Foundation)**: For building the application's user interface.
- **C#**: For implementing game logic and interactivity.
- **.NET Framework**: Provides the foundation for the application's functionality.
